# 기능 및 화면구성

## 기기 스크린

✔️ 장소별로 기기를 그룹화하여 관리할 수 있음  
✔️ 장소 내 기기를 모아볼 수 있음  
✔️ 장소를 추가할 수 있음  
✔️ 기기 고유 번호(시리얼넘버)를 직접 입력하거나 QR 코드 스캔을 하여 기기를 등록할 수 있음
